Practice Git Hub

  • Git is a open source distributed version control system which helps in tracking changes in code, collaborating with teams, and managing different versions of a project efficiently.

Git Commands

  • git init - Which will initialize the local repository on the system for that project
  • git status - It will help to check the status of the file
  • git add . - To move the file to the staging area
  • git commit -m "File added" - This will commit the changes to the local repository with the message
  • git log - view the commit history
